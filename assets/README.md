# Profile Photo Setup

Untuk menggunakan foto profil, silakan letakkan file foto Anda dengan nama `image_titan.png` di folder `assets/`.

## Format yang Disarankan:
- **Format**: PNG, JPG, atau WEBP
- **Ukuran**: Minimal 500x500px (persegi)
- **Resolusi**: 72-300 DPI
- **Ukuran file**: Maksimal 2MB

## Cara Mengganti Foto:
1. Letakkan foto Anda di folder `assets/`
2. Rename foto menjadi `image_titan.png`
3. Refresh halaman website

## Fallback:
Jika foto tidak ditemukan, akan tampil inisial "AP" sebagai fallback.

## CSS yang Diterapkan:
- Border radius 50% (bulat)
- Glowing border effect
- Rotating border animation
- Responsive sizing untuk semua device
- Object-fit cover untuk menjaga proporsi foto

Foto akan otomatis ter-crop menjadi lingkaran dan mendapat efek futuristik yang sesuai dengan tema website.
